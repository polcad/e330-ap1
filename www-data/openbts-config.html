OpenBTS Configuration Database

This page displays the actual configuration of your OpenBTS. If the key is "Static" you have to restart OpenBTS for the changes to take effect. It is recommended to use provided script to edit the database. You will not be able to start OpenBTS if your database is corrupted or configuration is invalid! This database is located in /etc/OpenBTS/OpenBTS.db

This example is not working because there is no SQLite3 connection available. See the corresponding PHP script in order to configure the connection.
