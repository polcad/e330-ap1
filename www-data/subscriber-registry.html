Subscriber Registry Configuration Database

This page displays the actual configuration of your Subscriber Registry Database. The Home Location Register (HLR) is a database that stores information about the user, including the user profile (preferences), account status, features and capabilities. 

The information stored includes:
• Subscriber identity (i.e. IMSI, MSISDN, IMEI/Hardware serial number)
• Subscriber supplementary services
• Subscriber location information
• Subscriber authentication information
If you see an IMSI number but there is no callerid associated with that number then this handset can make phone calls, but it will not be possible to receive any calls in most cases. The handset can however receive text messages send from CLI. It is recommended to use provided script to edit the database. For clarity only some columns of the database are displayed on this page. For a complete view of the database please use a database editor, i.e. sqlitebrowser. You will not be able to start OpenBTS or place a phone call if your database is corrupted or configuration is invalid! This database is located in /var/lib/asterisk/sqlite3dir/sqlite3.db
